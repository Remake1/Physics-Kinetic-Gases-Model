## About
Molecular kinetic theory simulation of gas particles in a container.
- Ideal gas
- Mixture of gases

## Project Setup

```sh
yarn install
```

### Compile and Hot-Reload for Development

```sh
yarn dev
```

### Compile and Minify for Production

```sh
yarn build
```
